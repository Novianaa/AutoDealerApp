# AutoDealerApp

AutoDealerApp adalah aplikasi web untuk manajemen data kendaraan bekas pada perusahaan otomotif.

## Tech Stack
- React.js
- Node.js
- Express.js
- MySQL
- Tailwind CSS

## Features
- Create product
- Get products (+ search by brand)
- Get detail product
- Update product
- Delete product

## Installation

### Clone repository
```bash
git clone https://github.com/Novianaa/AutoDealerApp.git
```

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm start
```
